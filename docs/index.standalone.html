CH. 01 / 06

为什么 Why

无限的窗口,无限的念头。"切去 App 记一笔"的成本,常常已经超过念头本身。

一个念头既然值得留下,
就不该再为记录它多付一次切窗口的成本。

所以 Memento 不是又一个新窗口——而是潜伏在系统层的一组右键服务和截图监听。让记录、回顾、总结都不离开你已经在的窗口。

它也尝试在克制的边界里调动机器的感官:每天只在第一次成功记录时,用前置摄像头、时间和当时天气留下一份现场证据;完成后立即退场,不影响当天后续记录。

CH. 05 / 06

喂给任何 AI Feed Any AI

一天结束,所有碎片聚成一份 markdown。三种姿势:

  • 拖文件夹
    把 ~/AISecretary 整个拖进 Claude 对话框。它先读 README 了解结构,再回答你"总结这周"、"最近反复出现了哪些主题"。
  • 一键复制
    Chrome dashboard 大按钮一点,今天的 md 进剪贴板。切到 ChatGPT、Gemini、Kimi、豆包,⌘V 即可。
  • Obsidian
    "Open folder as vault" 选 ~/AISecretary,获得全文搜索、双向链接、图谱视图。注意:不是来写,是来看。
